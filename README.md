# Digital Marketing Campaign Analysis | Python & Power BI

Analysis of digital marketing campaign performance across 8,000 customers using Python and Power BI.

## 📊 Key Metrics
- **Total Customers Analyzed:** 8,000
- **Total Ad Spend:** €40M
- **Overall Conversion Rate:** 87.6%
- **Best Channel:** Referral (highest conversions)
- **Most Cost-Efficient:** PPC (lowest cost per conversion)

## 🔍 Analysis Includes
- Channel performance comparison (Referral, PPC, SEO, Email, Social Media)
- Cost per conversion analysis
- Campaign type effectiveness (Awareness, Consideration, Conversion, Retention)
- Conversion rate by customer age and income
- Interactive Power BI Dashboard

## 🛠 Tools Used
`Python` `Pandas` `Matplotlib` `Seaborn` `Power BI`

## 📁 Files
- `marketing_campaign_analysis.ipynb` — Python analysis
- `marketing-campaign-dashboard.pbix` — Power BI dashboard
- `digital_marketing_campaign_dataset.csv` — Raw dataset
- `marketing_campaign_clean.csv` — Cleaned dataset
- `marketing_campaign_dashboard.pdf` — Dashboard export
